## Setup Instructions

### 1. Install Ollama

Download and install Ollama from [ollama.ai](https://ollama.ai)

### 2. Pull a Model

```bash
ollama pull mistral
# Or other models like: llama2, neural-chat, dolphin-mixtral
```

### 3. Start Ollama Service

```bash
ollama serve
```

This will start Ollama on `http://localhost:11434`

### 4. Set Up Python Environment

```bash
# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

## Running the Application

### Option 1: Direct Python Execution (Testing)

```bash
python mental_health_chatbot.py
```

This runs the chatbot in standalone mode for testing.

### Option 2: Streamlit Web UI (Recommended)

```bash
streamlit run streamlit_app.py
```

The app will be available at `http://localhost:8501`
